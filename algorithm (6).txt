ALGORITHM: Simple Payroll Management System

Step 1: Start the program
Step 2: Display welcome message "SIMPLE PAYROLL MANAGEMENT SYSTEM"
Step 3: Declare variables:
        - number Of Employees (int)
        - Employee objects array/vector
        - total Payroll (double)
Step 4: Prompt user: "Enter number of employees: "
Step 5: Read number Of Employees
Step 6: IF number Of Employees <= 0 THEN
           Display error message and ask again
        END IF
Step 7: FOR employee Index = 1 TO number Of Employees DO
           Step 7.1: Display "Employee X" (where X is current number)
           Step 7.2: Prompt for employee ID and store
           Step 7.3: Prompt for employee name and store
           Step 7.4: Prompt for department and store
           Step 7.5: Prompt for basic salary and store
           Step 7.6: Prompt for allowance and store
           Step 7.7: Prompt for deduction and store
           
           Step 7.8: VALIDATE inputs:
                    IF basic Salary < 0 OR allowance < 0 OR deduction < 0 THEN
                       Display "Invalid payroll record - negative values not allowed"
                       Skip this employee
                    ELSE
                       Step 7.9: Calculate gross Salary = basic Salary + allowance
                       Step 7.10: Calculate net Salary = gross Salary - deduction
                       
                       Step 7.11: DETERMINE salary status:
                                 IF deduction > gross Salary THEN
                                    Salary Status = "Review Required"
                                 ELSE
                                    Salary Status = "Ready for Payment"
                                 END IF
                       
                       Step 7.12: Display employee payroll report on screen
                       Step 7.13: Save employee payroll record to file
                       Step 7.14: Add net Salary to total Payroll
                    END IF
        END FOR
        
Step 8: Display summary:
        - Total employees processed: number Of Employees
        - Total payroll amount: total Payroll
        
Step 9: Display "Payroll report saved successfully to payroll_report.txt"
Step 10: End the program
